# Companies

One file per company you've worked at. This is the deep, raw, source-of-truth layer — every project, every win, every quote, every learning. Other docs (`core-profile/career-history.md`, the CV) summarize from here.

## How to add a company

1. Copy `_template.md` to `<company-name>.md` (lowercase, hyphenated).
2. Run the `interview` skill on the new file to fill it in via guided dialogue.

## When to delete this README

Once you have at least one real company file in this folder, delete this README. It only exists to onboard you to the structure.
